## 五、TemplateMatcher

```kotlin
object TemplateMatcher {
    fun match(screenBitmap, templateBitmap, threshold): MatchResult {
        Imgproc.matchTemplate(screenMat, templateMat, resultMat, TM_CCOEFF_NORMED)
        val minMaxLoc = Core.minMaxLoc(resultMat)
        // 返回全局最佳匹配坐标
    }
}
```

- 匹配算法: `TM_CCOEFF_NORMED`（归一化相关系数匹配）
- 返回: 全局最佳匹配的坐标（minMaxLoc）
- 阈值判定: `similarity >= threshold` 为匹配成功
- 注意: 全局匹配意味着 `JOIN_BUTTON` 可能匹配到画面中任意位置的按钮，不保证属于特定悬赏
