# FightNode - 战斗模块规格

## 屏幕状态


| 方向 | ScreenState | 说明                                |
| ---- | ----------- | ----------------------------------- |
| 进入 | FIGHT       | 从 BattleLoadingNode 加载完成后进入 |
| 退出 | SETTLEMENT  | 连续3次未识别跳跃，进入结算         |

## 交互动作

### 阶段一：下滑阶段


| # | 动作         | 裁剪区域       | 匹配模板     | 坐标/偏移             | 延迟   | 条件                    |
| - | ------------ | -------------- | ------------ | --------------------- | ------ | ----------------------- |
| 1 | 等待开场     | -              | -            | -                     | 6000ms | 进入节点后先等6秒       |
| 2 | 点击下滑按钮 | 左下 1/4       | SLIDE_BUTTON | X不变，Y偏移 rows*3/4 | 300ms  | 下滑阶段循环            |
| 3 | 点击血咒技能 | 左1/2 × 下1/4  | BLOOD_CURSE  | X不变，Y偏移 rows*3/4 | -      | 血咒出现时（仅点击1次） |

### 阶段二：下滑结束后检测上翻


| # | 动作     | 裁剪区域   | 匹配模板  | 坐标/偏移                        | 延迟 | 条件             |
| - | -------- | ---------- | --------- | -------------------------------- | ---- | ---------------- |
| 4 | 检测上翻 | 右下 1/4   | SCROLL_UP | X偏移 cols*3/4，Y偏移 rows*3/4  | -    | 下滑结束后立即检测 |

### 阶段三：Boss战阶段


| # | 动作         | 裁剪区域 | 匹配模板       | 坐标/偏移                                     | 延迟           | 条件                   |
| - | ------------ | -------- | -------------- | --------------------------------------------- | -------------- | ---------------------- |
| 5 | 检测Lv图标   | 左上 1/8 | LV_ICON        | -                                             | 100ms          | 快速扫描，Lv未出现时   |
| 6 | 点击大招     | 左侧 1/6 | ULTIMATE_SKILL | 识别则点击，未识别跳过                       | 无额外延迟     | 大招出现时（优先检测） |
| 7 | 点击跳跃     | 右下 1/4 | JUMP_BUTTON    | X偏移 cols*3/4，Y偏移 rows*3/4               | -              | 大招未识别时检测       |
| 8 | 点击武器技能 | 下方 1/4 | WEAPON_SKILL   | X不变，Y偏移 rows*3/4，**记住坐标，连点10次** | 240ms/次       | 武器技能出现时         |

## 裁剪区域说明


| 区域名       | 裁剪方法                              | 占全屏比例                  | 用途           |
| ------------ | ------------------------------------- | --------------------------- | -------------- |
| 左下 1/4     | `detector.cropBottomLeftFourth(mat)`  | 宽25% x 高25%（左下角）     | 下滑按钮       |
| 左1/2 × 下1/4 | `detector.cropBottomLeftHalf(mat)`   | 宽50% x 高25%（左下半）     | 血咒技能       |
| 左上 1/8     | `detector.cropTopLeftEighth(mat)`     | 宽50% x 高12.5%（左上）     | Lv图标检测     |
| 右下 1/4     | `detector.cropBottomRightFourth(mat)` | 宽25% x 高25%（右下角）     | 跳跃按钮、上翻(仅下滑后) |
| 左侧 1/6    | `detector.cropLeftSixth(mat)`         | 宽16.7% x 高33%（左侧中央） | 大招图标       |
| 下方 1/4     | `detector.cropBottomQuarter(mat)`     | 宽100% x 高25%（底部）      | 武器技能       |

## 常量定义


| 常量                    | 值     | 说明                                        |
| ----------------------- | ------ | ------------------------------------------- |
| SLIDE_INTERVAL_MS       | 300ms  | 下滑阶段扫描间隔                            |
| BOSS_DETECT_INTERVAL_MS | 100ms  | Boss未出现时快速扫描间隔                    |
| BOSS_LOOP_INTERVAL_MS   | 1000ms | Boss出现后常规循环间隔                      |
| MAX_SLIDE_MISS          | 3      | 下滑阶段连续无匹配次数，超过后进入Boss阶段  |
| MAX_JUMP_MISS           | 3      | 跳跃连续无匹配次数，超过后认为战斗结束       |
| MAX_SKILL_CLICKS        | 10     | 武器技能最大连点次数（大招无限制）           |
| SKILL_CLICK_INTERVAL_MS | 240ms  | 武器技能连点间隔                           |

## 决策逻辑

```
进入 → 等待6秒开场动画

下滑阶段（300ms间隔）:
  → 匹配 SLIDE_BUTTON（左下1/4）→ Y偏移点击，重置miss计数
  → 匹配 BLOOD_CURSE（左1/2 × 下1/4）→ Y偏移点击一次（仅首次）
  → 连续3次无匹配 → 下滑结束

下滑结束后:
  → 立即截图检测 SCROLL_UP（右下1/4）→ 有则点击，无则跳过

Boss阶段:
  未检测到Lv时（100ms快速扫描）:
    → 匹配 LV_ICON（左上1/8）→ Lv出现，进入战斗循环

  Lv出现后（1000ms常规循环）:
    → 匹配 ULTIMATE_SKILL（左1/6）→ 识别则点击，未识别则跳过（优先检测）
      → 大招出现期间不检测跳跃/武器
      → 大招消失后跳过一轮再检测跳跃/武器
    → 匹配 JUMP_BUTTON（右下1/4）→ X+Y偏移点击（大招未识别时才检测）
    → 匹配 WEAPON_SKILL（下方1/4）→ Y偏移记住坐标，连点10次，每次240ms
    → 连续3次未匹配 JUMP_BUTTON → 返回 SETTLEMENT
    → 30s 无匹配 → checkNodeTimeout
```

## 技能机制

**大招**：每轮都做模板匹配，识别到则点击，无额外延迟。大招出现期间（`ultimateActive=true`）不检测跳跃/武器，大招消失后跳过一轮再放行跳跃检测。无连点次数限制。
**武器技能**：记住坐标连点，第一次匹配后在记住的坐标上连续点击10次，每次间隔240ms，连点期间不做模板匹配。
**上翻**：仅在下滑结束后立即检测一次，Boss阶段不再检测。

## 坐标偏移说明

裁剪区域的匹配坐标相对于裁剪起点，需要偏移到全屏坐标：


| 操作      | 裁剪起点                       | X偏移    | Y偏移    |
| --------- | ------------------------------ | -------- | -------- |
| 下滑      | 左下1/4 (0, rows*3/4)          | 0        | rows*3/4 |
| 血咒      | 左1/2×下1/4 (0, rows*3/4)      | 0        | rows*3/4 |
| 上翻      | 右下1/4 (cols*3/4, rows*3/4)   | cols*3/4 | rows*3/4 |
| 跳跃      | 右下1/4 (cols*3/4, rows*3/4)   | cols*3/4 | rows*3/4 |
| 大招      | 左侧1/6 (0, rows/3)            | 0        | rows/3   |
| 武器技能  | 下方1/4 (0, rows*3/4)          | 0        | rows*3/4 |

## 异常处理


| 异常                   | 处理                      |
| ---------------------- | ------------------------- |
| 截图返回 null          | 等待当前间隔后重试        |
| 30s 无匹配             | checkNodeTimeout 超时检测 |
| 连续3次未识别跳跃      | 战斗结束，返回 SETTLEMENT |
